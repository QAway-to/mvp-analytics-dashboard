# mvp-analytics-dashboard

> Client analytics dashboard — KPI cards, trend charts, and activity feed in a single-page demo.

A Next.js UI that presents key business metrics for a sample client (Acme Corp). Demonstrates a clean dashboard layout with conversion rate trend, user activity chart, and a live events feed — all driven by mock data, ready to swap in a real API.

## Features

- **KPI cards** — conversion rate, active users, revenue, and sessions at a glance
- **Trend charts** — conversion and activity plotted over the last 30 days
- **Activity feed** — chronological event log with type labels
- **Client branding** — configurable client name and report period in the header

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | Next.js (Pages Router) |
| Styling | CSS Modules |
| Charts | Custom SVG components |
| Data | Mock JSON (drop-in API-ready) |

## Getting Started

```bash
npm install
npm run dev   # http://localhost:3000
```
